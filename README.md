# Multi-Step Form

A responsive multi-step subscription form built with **HTML5, CSS3, and Vanilla JavaScript (ES Modules)**.

This project is based on the Frontend Mentor Multi-Step Form challenge and focuses on creating a clean user experience while practicing JavaScript state management and modular architecture.

## ✨ Features

- Responsive design
- Four-step form navigation
- Form validation
- Monthly / Yearly billing toggle
- Dynamic plan pricing
- Add-on selection
- Live order summary
- Total price calculation
- Confirmation screen
- Modular JavaScript architecture (ES Modules)
